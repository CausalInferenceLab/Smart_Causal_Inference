markdown 예시#

Jupyter Book은 markdown 문서를 지원합니다.

아래와 같은 예시 코드를 입력하면 markdown 문법이 적용됩니다.

​```

This is an

tag#

This is an

tag#

This is an

tag#

This text will be italic This will also be italic

This text will be bold This will also be bold

You can combine them

  • Item 1

  • Item 2

    • Item 2a

    • Item 2b

  1. Item 1

  2. Item 2

  3. Item 3

    1. Item 3a

    2. Item 3b ​```
